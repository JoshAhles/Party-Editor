# Party-Editor (hosted invitation personalizer)

Static GitHub Pages site serving the **public, invitation-only** personalizer for each party
theme, at `editor.codebucky.com/<theme>/`.

- These pages are **client-side only** — buyers type details and the invitation PDF/image is
  generated in their browser. **No data is uploaded or stored** anywhere.
- This is the **lean taster** build (invitation only). The full personalization editor (all
  sheets + everything-included gallery) ships *inside* the purchased kit zip — it is not here.
- **Do not** add the kit zips, the full editor, or any pipeline source to this repo. This repo
  is intentionally limited to the public invitation editor.

Generated by `kit-pipeline/make_editor.mjs --public`. Don't hand-edit `*/index.html`.
